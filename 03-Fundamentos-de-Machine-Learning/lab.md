# Lab — Módulo 03: Fundamentos de Machine Learning

## Identificando o tipo certo de problema de ML

Treinar a leitura de problemas de negócio para mapear corretamente a técnica principal de ML.

## Pré-requisitos

- Ler o README do módulo.
- Separar exemplos de negócio do seu contexto.

## Passo a passo

1. Escreva três problemas: um de previsão, um de valor contínuo e um de agrupamento.
2. Marque para cada um se a saída desejada é classe, número ou grupo.
3. Associe classificação, regressão ou clustering.
4. Liste qual dado seria necessário e se existem rótulos.
5. Revise se algum caso poderia ser resolvido por serviço pronto em vez de ML customizado.

## O que observar

- A maior parte dos erros em prova nasce da leitura errada do objetivo do negócio.
- Rótulo disponível é um grande sinalizador para aprendizado supervisionado.

## Custos e limpeza

- Lab sem custo obrigatório se mantido em papel ou editor local.
- Se abrir serviços AWS, encerre recursos após o teste.
