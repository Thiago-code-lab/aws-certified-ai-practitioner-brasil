# Módulo 17 — Glossário

Este glossário resume os termos mais recorrentes no AWS Certified AI Practitioner. Use-o como revisão rápida antes dos simulados e na reta final.

## Como usar

- Leia o termo e tente explicar com suas próprias palavras.
- Revise primeiro os termos que misturam serviço, conceito e arquitetura.
- Volte aos módulos temáticos quando algum termo ainda parecer abstrato.

## Glossário essencial

| Termo | Nome por extenso | Definição prática |
| --- | --- | --- |
| A2I | Amazon Augmented AI | Revisão humana seletiva em workflows de IA. |
| Bedrock | Amazon Bedrock | Serviço gerenciado para consumo de foundation models. |
| Chunking | Divisão de documentos | Separação de documentos em partes menores para retrieval. |
| Embedding | Representação vetorial | Conversão de conteúdo em vetor para busca semântica. |
| Few-shot | Prompt com exemplos | Técnica de prompt que usa exemplos de entrada e saída. |
| Fine-tuning | Ajuste do modelo | Adaptação do comportamento do modelo com dados adicionais. |
| FM | Foundation Model | Modelo base reutilizável para várias tarefas. |
| Grounding | Ancoragem em contexto | Uso de fontes confiáveis para apoiar a resposta do modelo. |
| Guardrails | Controles de política | Mecanismos para restringir e proteger interações de IA. |
| Hallucination | Alucinação | Resposta plausível, mas incorreta ou inventada. |
| Inference | Inferência | Uso do modelo treinado para responder a novas entradas. |
| Knowledge Base | Base de conhecimento | Fonte documental conectada ao modelo para grounding. |
| LLM | Large Language Model | Modelo fundacional focado em linguagem natural. |
| PII | Personally Identifiable Information | Dado pessoal identificável. |
| Engenharia de prompts | Engenharia de prompts | Estruturação de instruções e contexto para orientar a saída. |
| RAG | Retrieval-Augmented Generation | Recuperação de contexto seguida de geração. |
| SageMaker AI | Plataforma de ML | Serviço de ponta a ponta para workflows de ML. |
| Token | Unidade de processamento | Fragmento de texto usado pelo modelo em entrada e saída. |
| Training | Treinamento | Fase em que o modelo aprende com dados. |
| Vector database | Base vetorial | Armazena embeddings e permite busca por similaridade. |

## Dicas de revisão

- Termos de GenAI costumam aparecer misturados em uma mesma questão: prompt, grounding, RAG, embedding e base vetorial.
- Confusões clássicas: Bedrock vs SageMaker AI, embedding vs geração, RAG vs fine-tuning, Guardrails vs IAM.
- Use os [flashcards](./flashcards.md) para revisão ativa.
